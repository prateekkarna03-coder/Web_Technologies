Step 1 of 5

Setting Up Your Development Environment

What is PHP?

PHP runs on your computer's server side. HTML runs in the browser. We're setting up PHP first.

What to Install:

  1. PHP - The language for server-side code
  2. Code Editor - VS Code or Sublime Text
  3. Web Server - PHP built-in server (recommended)
# Check if PHP is installed (Windows Command Prompt) php -v # Check PHP version and modules php -m
✓ Tip: Use PHP's built-in server for learning. It's easy and works on any computer.
⚠ Important: Make sure PHP is added to your system PATH so you can run it from anywhere in the command line.
Step 2 of 5

Creating and Configuring PHP Files

How to Create a PHP File

PHP files look like text files but run on the server. Create a new file and name it index.php.

Basic PHP File Structure:

Create a file named index.php with the following structure:

<?php // This is a PHP comment $greeting = "Hello, World!"; echo $greeting; ?>

Project Structure Example:

your-project/ ├── index.php ├── config.php ├── css/ │ └── style.css ├── js/ │ └── script.js └── images/
✓ Tip: Keep folders organized. It makes your code easier to find later.
⚠ Important: PHP files must have the .php extension to be processed by the PHP server. Never use .html for files containing PHP code.
Step 3 of 5

Integrating PHP with HTML

How PHP & HTML Work Together

PHP: Runs on the server → HTML: Sent to browser → Result: User sees a webpage

Example: Displaying Dynamic Content

<!DOCTYPE html> <html> <head> <title>My PHP Page</title> </head> <body> <h1><?php echo "Welcome!"; ?></h1> <p><?php echo date("Y-m-d H:i:s"); ?></p> </body> </html>

Key PHP Tools:

  1. echo - Print text on the page
  2. Variables - Store data ($name = "John")
  3. if/else - Make decisions
  4. loops - Repeat code
✓ Tip: Don't mix PHP and HTML too much. Keep PHP in separate files to stay organized.
Step 4 of 5

Running Your Application Locally

Run Your Project Locally

Use the built-in PHP server to test your project on your computer before uploading it.

Using PHP Built-in Server (Easiest Method):

# Navigate to your project folder cd C:\path\to\your-project # Start the built-in PHP server php -S localhost:8000 # Open your browser and go to: # http://localhost:8000

Using Apache Server:

  1. Place your project in Apache's root directory (htdocs)
  2. Start Apache using Apache Control Panel or command line
  3. Access your site via http://localhost/your-project
✓ Tip: Use built-in server for practice. Use Apache for real websites.
⚠ Important: If port 8000 doesn't work, try 8001 or 8080.
Step 5 of 5

Troubleshooting & Best Practices

Quick Fixes

Problem 1: "PHP is not recognized"

Fix: PHP is not installed or not in PATH. Download PHP and add it to your computer's settings.

Problem 2: "Port 8000 in use"

Fix: Use a different port: php -S localhost:8001

Problem 3: "500 Error"

Fix: Check your PHP code for typos. Look at the error message.

✅ Best Practices:

  • Test your code before uploading it
  • Write comments to explain your code
  • Use simple variable names like $name, $age
  • Keep passwords out of your code
  • Check what users type in (don't trust it)
  • Use Git to save your work
